The following document is for all customers using the Azure HANA Backup and Recovery scripts.
Please fill in all necessary fields for each SID you wish to backup or recover. You may also comment
out fields that you do not wish to backup for a period of time by adding a # in front of a required field.
You also do not need to enter all SIDs that are contained on a particular server if there is no
need to backup or recover that particular SID. The format below must be kept for all fields otherwise all
scripts will present an error message and the script will terminate.  You may however delete additional required
fields of any SID Information Details you will not be using after the last usable SID.  All fields must
either be filled in, commented out, or deleted.
